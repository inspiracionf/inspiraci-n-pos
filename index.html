👥 Crear Cuenta de Operador

Los operadores solo pueden registrar ventas e imprimir tickets. No tienen acceso a estadísticas, exportación ni este panel.

📂 Catálogo de Productos

Sube un archivo CSV con tres columnas: codigo, descripcion, precio (el precio es opcional). Ejemplo: COD-01,Rosas rojas x12,350

📂 Subir catálogo CSV — clic para seleccionar

🗑️ Gestión de Ventas

Para eliminar una venta individual, abre el detalle desde Ventas del Día y usa el botón rojo al fondo.

Para empezar de cero (borrar ventas de prueba), usa el botón de abajo. Se recomienda exportar CSV primero.
